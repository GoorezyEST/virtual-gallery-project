Hover Lights

In this project I aim for another challenge based on good aesthetics and creating a reusable effect in another project or website. I used pseudo elements for the development of this project, as well as some keyframes. Finally I added a small menu, which changes its display when the middle button is clicked.

SCSS was mainly used for the development of this small project. Although some necessary features were added by implementing TypeScript code.
